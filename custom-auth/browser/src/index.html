This is a simple guide for Lit Protocol custom auth

This guide outlines the process of implementing custom authentication using Lit Protocol, starting with minting programmable key pairs (PKP) and managing custom authentication methods. It includes steps to create and authorize custom auth methods, convert Lit Action codes to IPFS CIDs for smart contract interactions, and utilize session signatures to pkpSign and executeJs, methods provided in the litNodeClient.
